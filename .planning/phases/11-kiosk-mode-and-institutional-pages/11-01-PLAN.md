    3. Create `packages/web/src/hooks/useKioskTimer.ts`:
       - Import `useIdleTimer` from `react-idle-timer`
       - Constants: `IDLE_TIMEOUT_MS = 10 * 60 * 1000` (10 min), `PROMPT_BEFORE_MS = 60 * 1000` (60 sec)
       - Accept `enabled: boolean` parameter
       - Pass `disabled: !enabled` to `useIdleTimer`
       - `onPrompt`: set `showWarning = true`, start `setInterval` counting down `secondsLeft` from 60
       - `onIdle`: call `performLogout` — POST /api/auth/logout (use the existing `logout` from auth API client), then `queryClient.clear()`, then `window.location.href = "/"` (hard navigation, NOT `navigate()`)
       - `dismissWarning`: clear interval, set `showWarning = false`, call `reset()` from useIdleTimer
       - Cleanup: clear interval on unmount
       - Return `{ showWarning, secondsLeft, dismissWarning, performLogout }`
       - Use `useQueryClient()` to get the query client instance
       - Do NOT use `queryClient.invalidateQueries()` — use `queryClient.clear()` for complete wipe (KIOSK-03)
  </action>
