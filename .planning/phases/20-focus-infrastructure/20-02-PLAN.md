<objective>
Replace all non-compliant focus ring styles across the codebase with WCAG-compliant tokens. Standardize on `focus-visible:` prefix and `accent-500` color to match the global `*:focus-visible` rule and the shared Input/Button components.

Purpose: WCAG 2.4.11 requires focus indicators to have at least 3:1 contrast. The current `focus:ring-primary-500/30` (30% opacity) achieves only ~1.4:1. Additionally, using `focus:` instead of `focus-visible:` shows keyboard-style rings on mouse click, which is unnecessary.
Output: All 20 files with focus:ring patterns updated to use compliant, consistent tokens.
</objective>
