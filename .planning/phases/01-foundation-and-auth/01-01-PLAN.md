       - `contributorStatus` pgEnum: "onboarding", "active", "paused", "inactive"
       - `contributors` table: id (uuid PK), name (text, not null), email (varchar 255, unique, nullable), username (varchar 100, unique, nullable), phoneNumber (varchar 20, unique, nullable), passwordHash (text, nullable), authProvider (enum, not null), authProviderId (varchar 255, nullable), role (enum, default contributor), status (enum, default onboarding), createdAt, updatedAt, createdBy (self-reference, nullable)

       - Export all tables and enums
