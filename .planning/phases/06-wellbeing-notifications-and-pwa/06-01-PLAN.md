```typescript
export interface WellbeingCheckinInput {
  uclaItems: [number, number, number]; // each 1-4
  wemwbsItems: [number, number, number, number, number, number, number]; // each 1-5
  consentGranted: boolean; // must be true
}

export interface WellbeingCheckin {
  id: string;
  uclaScore: number;
  wemwbsScore: number;
  completedAt: string;
}

export interface WellbeingDueResponse {
  due: boolean;
  lastCheckinAt: string | null;
}

export interface WellbeingTrajectoryPoint {
  uclaScore: number;
  wemwbsScore: number;
  completedAt: string;
}
```

- wellbeingCheckinSchema: Zod object validating uclaItems (array of 3, each z.number().min(1).max(4)), wemwbsItems (array of 7, each z.number().min(1).max(5)), consentGranted (z.literal(true) — consent MUST be true to submit)

**6. Server routes** — Create `packages/server/src/routes/wellbeing.ts`:

`POST /api/wellbeing/checkin` (auth required):
1. Parse body with wellbeingCheckinSchema
2. Compute uclaScore = sum of uclaItems (3-12)
3. Compute wemwbsScore = sum of wemwbsItems (7-35)
4. Insert consent record: purpose='wellbeing_checkin', granted=true, policyVersion='1.0', ip and user-agent from req
5. Insert wellbeingCheckins row with all 10 item columns, both scores, consentRecordId, contributorId from auth
6. Return 201 with { id, uclaScore, wemwbsScore, completedAt }

`GET /api/wellbeing/due` (auth required):
1. Query most recent wellbeingCheckins for contributorId, order by completedAt desc, limit 1
2. If no rows: return { due: true, lastCheckinAt: null }
3. If most recent completedAt is > 56 days ago: return { due: true, lastCheckinAt }
4. Otherwise: return { due: false, lastCheckinAt }

`GET /api/wellbeing/history` (auth required):
1. Query all wellbeingCheckins for contributorId, order by completedAt asc
2. Return array of { uclaScore, wemwbsScore, completedAt }

**7. Update impact route** — In `packages/server/src/routes/impact.ts`, add wellbeingTrajectory to the GET /api/impact/summary response:
- Query wellbeingCheckins for the contributor, order by completedAt asc
